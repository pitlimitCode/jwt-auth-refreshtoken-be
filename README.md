## JWT Auth Refresh Token - Back-End

### Cloning to local:  
Direct your path in command-prompt to put this project, then:  
```
git clone https://github.com/pitlimitCode/jwt-auth-refreshtoken-be.git
```
```
cd jwt-auth-refreshtoken-be
```
```
npm install
```

run project:
```
nodemon
```
